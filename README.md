# Student Inventory Management System

A comprehensive student inventory system with Google Forms integration, automatic photo uploads, and role-based dashboards.

## Features

- Student registration and login
- Individual inventory form with photo upload
- Google Forms integration for data collection
- Automatic data sync to Supabase
- Separate dashboards for students and admin
- Organized data output in Google Sheets
- Professional UI with office branding

## Tech Stack

- Frontend: React + TypeScript + Vite
- Backend: Supabase (Auth, Database, Storage)
- Forms: Google Forms API
- Styling: Tailwind CSS
- File Upload: Supabase Storage

## Setup Instructions

1. Install dependencies: `npm install`
2. Configure Supabase credentials in `.env`
3. Set up Google Forms API credentials
4. Run development server: `npm run dev`
